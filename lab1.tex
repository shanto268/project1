    \hypertarget{theory}{%
\subsection{Theory}\label{theory}}

We the theoretical maximum power \(P_{theo} = 100 W\) and the maximum
open circuit voltage \(V_{oc} = 21.6 V\). Therefore, the theoretical
resistance for maximum power should be

\[ R_{theo} = \frac{{V_{oc}^2}}{P_{theo}} = 4.6656 \Omega  \]

    \hypertarget{methodology}{%
\subsection{Methodology:}\label{methodology}}

We are planning on varying the resistance source from short-circuit to
open-circuit with 0 to 10 Ohm resistors connected to the circuit.

For each resistor connected, we measure the current and voltage. We
calculate power using the following formula:

\[P = IV\]

We aim to repeat each reading 3 times to ensure accuracy of results and
plan to tabulate all our data. Using this tabular data we would create
the required \textbf{V vs I} and \textbf{V vs P} plots.
